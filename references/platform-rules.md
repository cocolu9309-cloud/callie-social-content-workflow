# Platform Rules

Use this reference when generating platform-specific Callie content. Adapt structure and intent by platform.

## Instagram

Use for visual storytelling, lifestyle scenes, carousel education, and Reels.

- Reels: 3-second visual hook, 15-30 second structure, short caption, selective hashtags.
- Carousel: strong first slide, 5-7 slides, gift guide or story progression.
- Stories: polls, question stickers, "this or that", countdowns, link CTA.
- Tone: polished, warm, visual, concise.

Include:

- Reel hook
- Caption
- Carousel or Story idea when relevant
- 5-8 hashtags max

## Facebook

Use for community-style conversation, family occasions, gifting decisions, and broader campaign copy.

- Slightly longer caption is acceptable.
- Add a question that invites comments.
- Good for holiday reminders, family stories, and gift guide posts.
- Avoid hashtag-heavy posts.

Include:

- Post copy
- Conversation prompt
- CTA

## YouTube Shorts

Use for short story arcs, gift reveals, list-style ideas, and "before the moment / after the reveal" structures.

- Title should be searchable and emotional.
- Script should include scene beats and on-screen text.
- Description can include keywords and CTA.
- Avoid relying on sound-only context.

Include:

- Shorts title
- 15-30 second script
- On-screen text
- Description
- Thumbnail text

## TikTok

Use for trend-aware, fast hooks, POVs, reveals, gift transformations, and relatable relationship moments.

- Start with a visual or text hook in the first 3 seconds.
- Use formats such as POV, "things that feel personal", "gift reveal", "if they like X", "before/after", and "I didn't know what to get until..."
- Suggest sound/trend type, but do not invent exact current audio unless verified.
- Keep caption short and natural.

Include:

- 3-second hook
- 15-30 second video script
- On-screen text
- Sound/trend suggestion
- Caption
- Hashtags

## Pinterest

Use for searchable, save-worthy, evergreen inspiration and gift planning.

- Prioritize keywords and visual clarity.
- Use vertical image/pin direction.
- Make content useful as a gift guide, moodboard, checklist, or occasion idea.
- Avoid overly casual slang.

Include:

- Pin title
- Pin description
- Keywords
- Board suggestion
- Vertical visual direction
- Save-worthy angle

## X

Use for concise observations, gifting questions, countdowns, polls, and light trend commentary.

- Keep posts short and conversation-friendly.
- Avoid too many hashtags.
- Threads should start with a sharp insight or question.
- Use trends only when Callie can add an authentic gifting or relationship angle.

Include:

- Short post
- Optional thread opener
- Poll or reply prompt when useful
- 0-2 hashtags max

## Website

Use for campaign landing pages, homepage banners, collection modules, and product/category storytelling.

- Copy must be clear and conversion-aware.
- Avoid social slang.
- Tie message to the occasion and category.
- Include CTA labels that fit shopping behavior.

Include:

- Banner headline
- Supporting line
- CTA
- Optional module copy
